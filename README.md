# M1OEP-cldiego

* Having trouble keeping track of clients, payments, invoices, etc? This program is going to make all of these easier for you.
The program was inspired by my hardships of using strictly google sheets to run the finances for my landscaping business 
C & D Lawn Care services. 

* This program prompts for user input with displaying a menu that has 5 options for the user to choose from, which all have inout validation within them. 
* The prgram adds clients to a csv file and assigns them a special id #, it keeps track of the clients name and their balance amount! Able to subtract from clients balnce by using negative symbols!

* In the future looking to expand on it more where you can  have the program create invoices for clients, ultimately make it more thorough and better!

* Good starting point for an awesome long term project!
