# Finance Tracker For Small Business #

* Having trouble keeping track of clients, payments, invoices, etc? This program is going to make all of these easier for you.
The program was inspired by my hardships of using strictly google sheets to run the finances for my landscaping business 
C & D Lawn Care services. 

* This program prompts for user input with displaying a menu that has 6 options for the user to choose from, which all have inout validation within them. 
* The program adds clients to a csv file and assigns them a special id #, it keeps track of the clients name and their balance amount! Invoice creation allowing
  for business owners to invoice clients quickly and efficiently. Template invoice document currently set up for my business but can easily be modified for
  other businesses!

## Languages/Extensions Used ##
* Primarily C++
* Used Python for Invoice Maker
* Template doc file
* .txt files
